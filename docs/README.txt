Install NETMAP on your ubuntu system - https://github.com/luigirizzo/netmap

To run the code, install ubuntu 14 and disable the gnome (gui). Open the teletypewriter and follow this guide to install cuda-6.4
https://www.insiderattack.net/2014/12/installing-cuda-toolkit-65-in-ubuntu.html.

After running installing cuda-6.4 and netmap run the Makefile.
